## рҹ’Ў The Story Behind It

Ever tried warming up before a coding session? Most typing tests throw random English at you вҖ” _"The quick brown fox jumps..."_ вҖ” and that's fine, but it doesn't prepare your fingers for the characters that actually matter: `{}`, `=>`, `//`, `</>`.

DevType was born out of that frustration. I wanted something that feels like a real IDE warm-up вҖ” not a grade-school typing class. So I built one.

---

## вңЁ Features at a Glance

| Feature | Description |
|:--------|:------------|
| рҹ–ҘпёҸ **Code-First Typing** | Practice with **JavaScript**, **Python**, **HTML**, **C++**, and **English** вҖ” real syntax, real logic |
| рҹ“Ҡ **Live Metrics HUD** | Watch your **WPM**, **accuracy**, and **errors** update in real-time as you type |
| вҸұпёҸ **Flexible Timers** | Choose between **15s**, **30s**, **60s**, or **120s** sessions to match your mood |
| рҹҺҜ **3 Difficulty Levels** | From gentle `console.log()` to async class architectures вҖ” pick your poison |
| рҹ”җ **Secure Auth System** | JWT tokens in httpOnly cookies + bcrypt hashing вҖ” your data stays yours |
| рҹ“Ҳ **Personal Dashboard** | WPM trends, language breakdowns, test history вҖ” track your growth over time |
| рҹҸҶ **Global Leaderboard** | See how you stack up against other developers worldwide |
| рҹҺЁ **Cyberpunk UI** | Dark theme with neon accents, glass panels, GSAP animations, and glow effects |
| вҢЁпёҸ **Smart Shortcuts** | Start typing to begin вҖў `Tab + Enter` to instantly restart |
| рҹ”„ **600+ Snippets** | Procedurally generated content means you rarely see the same snippet twice |

## рҹҡҖ Getting Started

### Prerequisites

Before you begin, make sure you have:

- **Node.js** `v18+` вҖ” [Download here](https://nodejs.org/)
- **MongoDB** connection string вҖ” [Get a free cluster](https://www.mongodb.com/atlas)

### вҡЎ Quick Setup

```bash
# 1пёҸвғЈ  Clone the repo
git clone https://github.com/yash23082007/dev-type.git
cd dev-type

# 2пёҸвғЈ  Install dependencies
npm install

# 3пёҸвғЈ  Set up your environment
#     Create a .env file in the root:
```

```env
DATABASE_URL="mongodb+srv://<username>:<password>@cluster0.mongodb.net/devtype-db?retryWrites=true&w=majority"
JWT_SECRET="your-super-secret-key-change-this-in-production"
```

```bash
# 4пёҸвғЈ  Generate Prisma client
npx prisma generate

# 5пёҸвғЈ  Seed the database with 600+ snippets
npx ts-node prisma/seed.ts

# 6пёҸвғЈ  Launch the dev server
npm run dev
```

Then open **[http://localhost:3000](http://localhost:3000)** and start typing! рҹҺү

## рҹ§  Design Decisions

- **Why Zustand over Redux?** вҖ” Minimal boilerplate. Two small stores handle everything without provider hell
- **Why GSAP over CSS animations?** вҖ” The screen-shake on typing errors needs programmatic control that CSS `@keyframes` can't easily provide
- **Why JWT in cookies?** вҖ” httpOnly cookies prevent XSS token theft. The middleware layer handles verification without client-side token management
- **Why procedural snippet generation?** вҖ” 5 languages Г— 3 difficulties Г— randomized vocabulary = 600+ unique snippets without manually writing each one
